============================================================
SMART FORKING - Quick Reference
Continue past Claude Code sessions with full context
============================================================

STEP 1: Open a Regular Terminal
-------------------------------
Not inside Claude Code - just a normal terminal window.


STEP 2: Search for Your Session
-------------------------------
    fork "your topic"           # Search by relevance
    fork-latest "your topic"    # Search by date (newest first)

Examples:
    fork "ops hub"
    fork "magnusview"
    fork "google drive"
    fork-latest "firebase"


STEP 3: Pick a Session
-------------------------------
You'll see results like:

    > 1. #279f79cc (Recommended)
         USER: i need to organize my files...
         Project: Ops-Hub-Portal | Score: 85% | Date: 2026-01-08

    > 2. #a994a33e
         ...

    Fork Commands:
      #1: claude --continue 279f79cc-a3fc-4de5-8bf3-d5ea65815fe1
      #2: claude --continue a994a33e-8bff-4d39-b97c-4175cf6bc6e9


STEP 4: Copy & Run the Command
-------------------------------
Copy the fork command for the session you want:

    claude --continue a994a33e-8bff-4d39-b97c-4175cf6bc6e9

Claude Code starts with full context from that session!


============================================================
QUICK COMMANDS
============================================================

    fork "topic"              Search sessions by relevance
    fork-latest "topic"       Search sessions by date
    fork --sort date "topic"  Same as fork-latest
    fork -n 10 "topic"        Show 10 results instead of 5

    claude --continue <id>    Start Claude with session context
    claude --resume           Resume your last session
    claude                    Start fresh


============================================================
MAINTENANCE
============================================================

Update the index after working (adds new sessions):

    cd ~/Desktop/smart-forking
    source venv/bin/activate
    python indexer.py
